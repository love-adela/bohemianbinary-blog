# Cookie Counter

## 가상환경 실행

```
$ source myvenv/bin/activate
```
 
## To run the application
역할
* debugger를 작동시킵니다.
* 자동 reloader를 작동시킵니다.
* Flask Application에서 debug를 가능하게 합니다.

```
$ export FLASK_ENV=development
$ flask run
```

[출처 : Flask Quick Start](http://flask.pocoo.org/docs/1.0/quickstart/)
